Vue学习——如何创建Vue项目

需要环境

  • Node.js
    去Node官网下载Node.j：Node.js下载
    下载node主要是用到它的npm命令
    当出现以下情况时，证明我们安装好了Node.js

  • 安装cnpm
    众所周知，npm在国内的下载速度可谓是比得上百度网盘不开会员下载（略微夸张了些，还是比网盘快很多的），因此我们使用淘宝镜像下载cnpm，加快下载速度。
    运行命令：
    npm install -g --registry=https://registry.npm.taobao.org

项目初始化

  • 安装vue-cli
    cnpm install @vue/cli -g //全局安装 vue-cli

  • 创建项目
    vue create 项目名称

  • 选择创建方式：默认或者自定义

  • 选择默认方式时：选择安装一些配置项（babel、router等）

  • 选择路由是否使用history模式

  • 是否保存为一个预设（一般为no）

至此，创建完成。
