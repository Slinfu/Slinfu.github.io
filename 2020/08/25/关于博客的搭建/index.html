关于个人博客的搭建

近日又重新搭建了一次个人博客，觉得有必要记录一下如何搭建一个个人博客。

需要环境

  • Node.js
    去Node官网下载Node.j：Node.js下载
    下载node主要是用到它的npm命令
    当出现以下情况时，证明我们安装好了Node.js

  • 安装cnpm
    众所周知，npm在国内的下载速度可谓是比得上百度网盘不开会员下载（略微夸张了些，还是比网盘快很多的），因此我们使用淘宝镜像下载cnpm，加快下载速度。
    运行命令：
    npm install -g --registry=https://registry.npm.taobao.org

  • 安装Hexo
    这里我主要是使用了Hexo去搭建博客，因为Hexo比较流行，在搭建博客过程中就算出错了，在网上也基本能找到解决方法
    运行命令：
    cnpm install -g hexo-cli

至此，所需环境搭建好了。

搭建博客

本地搭建
  1. 先新建一个文件夹用来存放我们的博客数据。
  2. 在文件夹路径下进入cmd命令，运行代码 hexo init 初始化文件夹。
  3. 执行 hexo s 命令启动。启动完成后即可在本地地址下查看。

至此，本地搭建完成。

部署到GitHub
  1. 在GitHub自己的账号下新建一个仓库，仓库名格式须为：个人github的用户名 + .github.io
  2. 在我们刚新建的文件夹路径下去下载 cnpm install --save hexo-deployer-git
  3. 修改文件夹路径下的_config.yml文件

    其中，repo: https://github.com/Slinfu/Slinfu.github.io.git 是自己的仓库地址。
  4. 部署远端。
    运行命令：
    生成命令：hexo g
    推送命令：hexo d

至此，部署完成

以上便是我在搭建个人博客中的具体操作。
